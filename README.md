# ⌨️ Key Codes Project

A simple and interactive web application that displays key codes, key names, and key values in real-time when a user presses any key on the keyboard.

---

## 📌 Features

- 🔑 Detects key press events instantly
- 📊 Displays:
  - Key (e.g., "A", "Enter")
  - Code
  - Key Code
- ⚡ Real-time updates
- 🎨 Clean and user-friendly UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript

---

## 📷 Demo

Press any key on your keyboard and see the details appear on the screen instantly.
